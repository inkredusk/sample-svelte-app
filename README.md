#### Sample app using Svelte

Demo contains: 
  * Configuration for ESLint and Prettier
  * Configuration for husky
  * Configuration for Commit Lint
 
#### Installation

```sh
yarn install
```

#### To test commit lint
```sh
yarn format
```

#### To test ESlint using Hooks
```sh
yarn format
```

#### To test Prettier 

```sh
yarn format
```
